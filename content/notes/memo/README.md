## メモ
- ポッドキャストをベースとしたデジタルガーデンは、僕にとって次のような意味を持つ。
 - SNSではできない未完成なメモを公開できる場所
 - 複数のデバイスをまたぐメモアプリ
 - ポッドキャストのアーカイブ置き場
 - 時系列性を排除した、概念ベースでのポッドキャスト配信サイト
- ポッドキャストから抽出された概念と、自分で手書きするノートが混ざることで、普通のSNSとは違う時間の流れができる。

## 仮説
- SNSではこんなにたくさん無造作に投稿できない
- メモアプリは飽きて突散らかしてしまうし僕のメモ以上のことが起きない
- 拡張性がありながら、無造作であることが許されるデジタルガーデンと、しゃべりっぱなしでいいポッドキャストの相性は良いのでは？
